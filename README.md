# Compress App

## Description

Compress App is a web application that allows users to upload and compress images and PDF files. Users can specify compression parameters, such as image width and quality for images, and size limits for PDF files. The app provides a simple and user-friendly interface, making it easy to reduce file sizes while maintaining quality.

### Features

- **Single File Upload**: Users can upload both images and PDF files in one form.
- **Custom Compression Options**: Users can specify compression settings, including:
  - **Image Compression**:
    - Width (in pixels)
    - Quality (percentage)
  - **PDF Compression**:
    - Size (in megabytes)
- **Dynamic UI**: The form adapts based on the selected file type, showing relevant options.
- **Responsive Design**: The app is styled for usability on various devices.

### Technologies Used

- **Backend**: Node.js with Express
- **File Upload Handling**: Multer
- **Image Compression**: Sharp
- **PDF Compression**: pdf-lib
- **Frontend**: HTML, CSS, JavaScript
